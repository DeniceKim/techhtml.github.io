html 요소는 HTML문서의 루트를 나타냅니다

작성자는 루트 html 요소에 lang 속성을 정의하여, 문서에 언어를 주기를 권장합니다. 이는 음성 합성 도구가 어떤 발음을 사용할 지, 번역 도구가 어떤 규칙을 사용할 지 등을 돕습니다.

It is recommended to keep the usage of attributes and their values defined on the html element to a minimum to allow for proper detection of the character encoding declaration within the first 1024 bytes.

아래 예제의 html 요소에 문서 언어를 영어로 설정하고 있습니다. 
<!DOCTYPE html>
<html lang="en">
  <head>
    <title>Swapping Songs</title>
  </head>
  <body>
    <h1>Swapping Songs</h1>
    <p>Tonight I swapped some of the songs I wrote with some friends, who
    gave me some of the songs they wrote. I love sharing my music.</p>
  </body>
</html>
